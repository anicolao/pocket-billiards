# Pocket Billiards

A modern, touchscreen-optimized pocket billiards (pool) game designed for large-format displays such as TV-sized tabletops. Experience the classic game of pool with intuitive touch controls and realistic physics.

## Overview

Pocket Billiards is a full-screen canvas-based billiards game that transforms large touchscreen displays into an interactive pool table. Whether you're setting up a game room, creating an interactive exhibit, or simply want a high-quality digital billiards experience, this game provides realistic physics and intuitive touch controls.

## Features

### Current Features
- **Touchscreen-First Design**: Optimized for large touchscreen displays with natural, intuitive controls
- **Full-Screen Canvas**: Immersive gameplay that utilizes the entire display
- **Modern Web Technologies**: Built with HTML5 Canvas for smooth, responsive graphics

### Planned Features
- **Realistic Physics**: Accurate ball collision, friction, and spin mechanics
- **Multiple Game Modes**: 8-ball, 9-ball, straight pool, and more
- **Touch Controls**: 
  - Drag to aim your shot
  - Pull back and release to control shot power
  - Apply spin with multi-touch gestures
- **Visual Feedback**: Clear aiming lines, power indicators, and ball trajectories
- **Score Tracking**: Keep track of games and statistics
- **Customization**: Adjustable table felt colors, ball sets, and difficulty settings
- **Multiplayer Support**: Local multiplayer for competitive play

## Requirements

### Hardware
- Large touchscreen display (recommended: 40" or larger)
- Modern device capable of running a web browser (PC, tablet, or embedded system)
- Multi-touch support recommended for advanced controls

### Software
- Modern web browser with HTML5 Canvas support:
  - Chrome 90+
  - Firefox 88+
  - Safari 14+
  - Edge 90+

## Getting Started

### Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/anicolao/pocket-billiards.git
   cd pocket-billiards
   ```

2. Open the game in your web browser:
   ```bash
   # If you have Python installed:
   python -m http.server 8000
   
   # Or use any other local web server
   ```

3. Navigate to `http://localhost:8000` in your browser

4. For touchscreen devices, ensure your browser is in full-screen mode (usually F11)

## How to Play

### Basic Controls
- **Touch and drag**: Aim your cue stick
- **Pull back and release**: Set power and shoot
- **Two-finger gesture**: Apply spin to the cue ball

### Game Rules
Follow standard pocket billiards rules for your chosen game mode. Detailed rule sets for each game mode will be provided in the game interface.
